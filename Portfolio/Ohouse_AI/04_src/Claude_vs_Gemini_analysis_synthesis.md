### **Phase 7: "What We Learned" (Failures & Iteration)**
**Claude's emphasis + Gemini's learning mindset**

```
What Didn't Work (And Why It Mattered)

Failure 1: Fully Automated Design
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Attempt: User input → AI generates design (no curation)
Result: 68% rejection rate
Users said: "Felt impersonal", "No choice"

Learning: Users want agency, not just personalization
         Balance = AI suggestions + human choice

Applied: Current system always shows 3 options + explains trade-offs

Failure 2: Real-Time 3D Rendering
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Attempt: Render room instantly as products added
Result: 12-second load time on mobile (too slow)
Users said: "Lag kills immersion"

Learning: Pre-computation beats real-time for experience
         Speed matters more than live updates

Applied: Pre-render library + dynamic recombination
        Load time: 12s → 0.8s

Failure 3: Async Feedback Loop
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Attempt: Users give feedback via text; AI iterates
Result: 5-day loop cycle (users forgot context)
Users said: "Lost track of what I was asking for"

Learning: Trust-building requires synchronous interaction
         Fast feedback > comprehensive but delayed

Applied: Daily 15-min sync with power users
        Feedback loop: 5 days → 1 day
        User retention: 35% → 68% (iteration participants)
```

**Why this works:**
- Failures show risk-taking (not playing it safe)
- Metrics prove learning (not just "lessons learned")
- Shows iteration velocity (improvement speed)
