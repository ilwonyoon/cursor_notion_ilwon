## 💻 실습 프로젝트 구조

### 프로젝트 A: AI 컬러 팔레트 생성기
```
난이도: ⭐⭐
학습: Claude API, 환경변수, API 호출
추천 대상: 처음 배우는 사람

구조:
Input (키워드 입력)
  ↓
API Call (Claude에게 요청)
  ↓
Display (색상 박스 표시)
  ↓
Copy (색상 코드 복사)
```

### 프로젝트 B: 타이포그래피 테스터
```
난이도: ⭐
학습: State 관리, Shadcn Slider, 실시간 반영
추천 대상: 가장 간단하게 시작하고 싶은 사람

구조:
Slider (폰트 크기, 행간, 자간)
  ↓
Real-time Preview (텍스트 변화 실시간 반영)
  ↓
Download JSON (개발자에게 전달)
```

### 프로젝트 C: AI 이미지 피드백 챗봇
```
난이도: ⭐⭐⭐
학습: Claude Vision API, 이미지 처리, 채팅 UI
추천 대상: 도전하고 싶은 사람

구조:
Upload Image (드래그 & 드롭 또는 클릭)
  ↓
Vision API Analysis (Claude가 이미지 분석)
  ↓
Chat Interface (질문하며 대화)
  ↓
Conversation History (대화 기록 저장)
```
